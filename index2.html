Propiedad Position

En HTML, la propiedad position se utiliza junto con CSS (Cascading Style Sheets) para controlar cómo se posicionan los elementos en la página web. Existen diferentes valores para esta propiedad, y cada uno define un comportamiento distinto en la ubicación de los elementos.

Aplicación de la propiedad position

Aquí están los principales valores de position:


  • static (valor predeterminado):

    • Los elementos se posicionan en el flujo normal del documento, sin alteraciones. No responden a propiedades como top, right, bottom, o left.

  • relative:

    • El elemento se posiciona relativamente a su posición original en el flujo del documento. Puedes ajustar su ubicación usando top, right, bottom o left, pero el espacio original que ocupaba sigue estando reservado.

  • absolute:

    • El elemento se posiciona absolutamente respecto a su contenedor más cercano que tenga un position distinto de static. Se mueve fuera del flujo del documento, lo que significa que no ocupa espacio y otros elementos se comportarán como si no existiera.

  • fixed:
    • Similar a absolute, pero el elemento se posiciona en relación a la ventana del navegador, por lo que permanece fijo en su lugar, incluso cuando se hace scroll en la página.
  • sticky
    • Combina los comportamientos de relative y fixed. El elemento se comporta como relative hasta que se alcanza un umbral definido (por ejemplo, hacer scroll hasta cierto punto), momento en el cual se vuelve fixed.
